# Portfolio Website

Personal portfolio for Ngoc Hui, a Computer Science student focused on full-stack development, AI-powered tools/features, and practical product-minded engineering. The site presents Jade's strongest development work first, while also showing a design background as a supporting strength for building better user experiences.

## Live Site

[jade-hui-portfolio.vercel.app](https://jade-hui-portfolio.vercel.app/)

## Website Content

### Home

The home page highlights a small set of featured development projects.

### Projects

The projects section is the main content of the site. It contains curated full-stack, AI-powered, and product-focused builds with stack details, project links, and full case study pages. Each case study explains the problem, approach, technical decisions, what could be improved, and what was learned.

Current featured projects include:

- Personalized Birthday Card
- PGAI Voice Bot Tester
- Clinic Appointment App
- Flashcards Generator

### Devlog

The devlog is a lightweight writing section for shorter, more frequent updates. It documents learning progress, coursework, build notes, and reflections from projects such as The Odin Project exercises. Devlog posts can include interactive demos, GitHub links, and short technical notes.

### Design

The design page is a secondary portfolio section for UX/UI work. It uses short cards that link out to Behance case studies instead of rebuilding full design case studies inside this site.

### About

The about page gives a short bio, and resume access.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- MDX content with gray-matter and next-mdx-remote
- Vercel deploy target

## Local Development

Install dependencies first:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```
