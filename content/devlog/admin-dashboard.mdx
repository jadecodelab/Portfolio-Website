I finished the Admin Dashboard project from The Odin Project's Intermediate HTML and CSS course.
The biggest focus was CSS Grid. I used it to organize the main page structure and to make the dashboard feel intentional instead of stacked together piece by piece. Flexbox still helped with smaller UI details, but Grid was the part that made the overall layout click.

I also got more practice with visual hierarchy. A dashboard has a lot of repeated information, so spacing, alignment, card sizing, and contrast matter more than I expected. Small choices made the page easier to scan: consistent gaps, clear section headings, and keeping similar items visually grouped.
