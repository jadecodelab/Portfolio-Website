Part of The Odin Project Foundations - a browser-based sketchpad built with vanilla HTML, CSS, and JavaScript.

The whole grid is generated dynamically with JavaScript. No static HTML, just a loop creating and appending elements to the DOM based on whatever size you pick - up to 100x100. That's where it clicked for me how DOM manipulation actually works in practice.
