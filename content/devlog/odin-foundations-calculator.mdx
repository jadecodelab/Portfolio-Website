## Note

I finished The Odin Project Foundations course and wrapped up the calculator project. This felt like an important checkpoint because the project brought together the basics I had been practicing separately: HTML structure, CSS layout, JavaScript functions, DOM events, and debugging.

The calculator was simple on the surface, but it made me slow down and think more carefully about state. I had to track the current number, the stored number, the selected operator, and what should happen when the user presses equals, clears the screen, or starts a new calculation.

## What I Practiced

This project helped me get more comfortable turning small pieces of logic into an interactive browser app. I also practiced reading my own code after something broke, tracing the flow of values, and making the interface respond in a way that felt predictable.
