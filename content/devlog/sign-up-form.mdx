Today I completed the Sign-up Form project from the Intermediate HTML and CSS course in The Odin Project.

This project focused on recreating a design using semantic HTML and CSS. I practiced building layouts with Flexbox, styling form inputs, working with focus and validation states, and paying closer attention to spacing and visual hierarchy.

One thing I learned is that spending time planning the HTML structure before writing CSS makes the layout much easier to build and maintain.

Another step forward in my journey to becoming a full-stack developer.
