## Problem

Students often have notes, slides, or study material, but turning that content into useful practice questions takes time. I wanted to build a tool that could shorten the path from raw notes to active recall without making the student manually write every card.

## Approach

I built Flashcards Generator as an AI-powered study app that turns notes into flashcards and supports the review flow after generation. The app supports file uploads, review mode, shuffling, and locally saved decks so the generated content can become an actual study session instead of a one-time output.

## Tech Decisions

Node.js and Express provided a straightforward backend for handling requests and connecting to the OpenAI API. JavaScript kept the frontend and backend workflow approachable while I focused on the user flow: input notes, generate cards, review them, and keep decks available locally.

## What I'd Improve

I would improve the quality controls around generated cards, such as letting users edit, delete, and rate cards before saving a deck. I would also add better organization for multiple subjects and make the review mode track progress over time.
